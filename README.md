# activitys-mobile

## 项目安装
```
npm install
```

### 测试环境
```
npm run serve
```

### 正式环境
```
npm run build
```

### 配置
```
1、@vue/cli 4.5.9 脚手架创建项目
2、postcss-px-to-viewport适配移动端
3、vant UI库
4、babel-plugin-import按需加载
```
